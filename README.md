# Jarvis Desktop Agent

A personal AI desktop assistant inspired by the AI used by Tony Stark.
Jarvis listens for the wake word **"Jarvis"**, understands voice commands, and controls your computer while displaying a hologram-style animated sphere on the desktop.

## Features

* Wake word detection ("Jarvis")
* Voice command recognition
* Computer control (open apps, folders, system actions)
* Voice responses
* Hologram-style animated sphere UI
* Real-time audio reaction animation

## Example Commands

Jarvis can understand commands like:

* "Jarvis open Chrome"
* "Jarvis open Telegram"
* "Jarvis open VS Code"
* "Jarvis shutdown the computer"
* "Jarvis open downloads folder"

## Project Structure

jarvis/

* main.py – main program runner
* config.py – configuration settings
* voice/ – voice recognition modules
* brain/ – command processing logic
* system/ – computer control functions
* ui/ – hologram sphere interface
